General settings :

  • Project's name: Defines the name of the archive which will be downloadable via the "Generate" button
  • Table's name : Generates a text file with SQL code which creates a table with this name and saves the output files with this name
  • Number of line : Defines the number of rows of data to generate
  • Generation seed : Allows you to choose a number which, if you use it again, will give you the same data during generation
  • Output format : Choose the output format (s) for the data

Data fields :

  • Name: name of the data, provides a header with this name in the CSV and / or a field of this name with values in a table in SQL

  • Data types :

    • Dictionary: allows you to choose from a list of available dictionaries. The dictionaries are related to a theme.
    • Numeric: allows you to randomly generate a number between a minimum and a maximum value
    • Formula: calculates values according to a formula of the model a * x [ + - * / or % for modulo] b where a, x and b can be column names or numbers.
  • Null: Number of rows of data that will not be defined, this value can be entered as a percentage or as an absolute value
  • Coding: correspondence system to match values to codes, such as "1; Man" will associate code 1 with the value "Man"

The different buttons :

  • Reset : reloads the page
  • Add / Remove row : allows adding / removing a data field
  • Add / Delete table : add a table in last position or delete the last table
  • Import an XML or SQL file: will allow you to preload the generation page via an SQL table creation or an XML configuration file
  • Generate : launches the generation script

The examples :

  • Will preload the page ready to be generated in order to show an example of simple data to generate.
